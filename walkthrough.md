# Walkthrough - Edge-TTS Synthesizer

This walkthrough explains how to use the Edge-TTS Synthesizer.

## Step 1: Launch the Application
Start the server by running:
```bash
streamlit run app.py
```
This will open a new tab in your default web browser.

## Step 2: Configure Voice Settings
In the sidebar on the left, you can:
- **Select Voice:** Choose from a variety of Microsoft Edge neural voices.
- **Speech Rate:** Adjust how fast the voice speaks. 0 is the default speed.
- **Pitch:** Adjust the tone of the voice. 0 is the default pitch.

## Step 3: Synthesize Speech
1. Enter your text in the large text area in the center.
2. (Optional) Provide a custom filename for the output MP3.
3. Click the **✨ Generate & Play** button.
4. Wait for the "Synthesis complete!" message.

## Step 4: Preview and Download
- After synthesis, an audio player will appear. Click play to listen.
- If you are satisfied, click **📥 Download MP3** to save the file to your computer.
